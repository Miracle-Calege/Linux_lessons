cd->usado para navegar entre directorios e pastas

cd ou cd ~ ->leva para o directorio home

cd /->leva para o directorio root

cd ..-> leva para o directorio pai do directorio actual(recua um directorio)

cd+relative path

ex:cd Documents

cd+ absolute path

ex: cd /home/miracle-calege/Documents

cd My\ books->resultara

cd "My books"->resultara
